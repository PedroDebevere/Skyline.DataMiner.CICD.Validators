# CheckOptionsAttribute

## UnknownOption

### Details

The timer options are formatted as follows: \<optionName\>:\<optionValue\> where \<optionName\> should be one of the following:  
\- each  
\- dynamicThreadPool  
\- ignoreIf  
\- instance  
\- ip  
\- ping  
\- pollingRate  
\- qaction  
\- qactionBefore  
\- qactionAfter  
\- threadPool  
Options should be separated using semicolons (';').
